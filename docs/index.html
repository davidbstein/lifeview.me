Your life, at a glance

A beautiful dashboard for your home.

This totally free, MIT-licensed open-source project allows you to collect all of the important information about your day and displays it on a dashboard in your home

Respecting your privacy.

all software runs on the device itself, no data is collected. In fact, we don't even have a server, and this webpage is completely static and also does not collect information.

Supported Data Sources:

  • Time and Task Trackers:
    • Timeular
    • Clockify
    • Asana
    • Trello
    • Jira
    • Github Issues
    • Google Keep
  • Calendars
    • Google Calendar
    • Outlook
    • iCalendar
  • Fitness Trackers
    • Strava
    • Peloton
    • Fitbit
    • MyFitnessPal
  • Other Services
    • Weather
    • Alexa voice control
    • Sunrise / Sunset
    • RSS News Readers
Are we missing one? Send a pull request or modify the source code of the dashboard yourself!
